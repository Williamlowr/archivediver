# Web App

React + Vite frontend for ArchiveDiver.

## Purpose

- Collect topic, optional period, and artifact count.
- Render exhibit title, intro, artifacts, timeline, and dev details.
- Call backend API only.

## Local Run

- `npm install`
- `npm run dev`

## Environment

- `API_BASE_URL` (points to `apps/api`)

## Test

- `npm run test`
